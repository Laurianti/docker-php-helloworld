# Setup

 - Ensure you have Docker installed
 - `git clone` this repository
 - `sudo docker build -t docker-php-helloworld .` 
 - `sudo docker run --privileged -p 12080:80 docker-php-helloworld`
 - Go to http://127.0.0.1:12080/
